== Description ==

[OmniKick](https://www.omnikick.com) is a standalone application that integrates all web platforms including WordPress.
You must have an account in order to use this plugin. [Register here](https://www.omnikick.com/sign-up/)

This plugin's data will be handled via the [OmniKick](https://www.omnikick.com) SAAS platform.

Features:

* Optins
* Push Notification
* Analytics


== Installation ==

1. Search on plugin page **omnikick** or Upload `omnikick` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
